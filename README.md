# Latency Monitor

A real-time network latency monitoring tool with smooth animated graphs.

## Live Demo

**[View Live](https://cdn.jsdelivr.net/gh/PazerOP/ping@claude/fix-animation-timing-7zUC6/index.html)**

## Features

- Real-time ping measurements to popular endpoints (Google, Cloudflare, Amazon, GitHub, Apple)
- Smooth animated graph with bezier curves
- Configurable polling intervals (500ms - 5s)
- Statistics: current, min, max, average latency, jitter, and packet loss
- Packet loss visualization with red X markers
- Request log with color-coded response times

## Usage

Open `index.html` in any browser, select a target and interval, then click Start.
